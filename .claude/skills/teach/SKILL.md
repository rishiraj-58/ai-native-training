# Teach Skill

Teach the requested concept following these rules:
1. Start with ONE core idea only
2. Give a practical code example (Java preferred)
3. Ask a comprehension question before proceeding
4. After correct answer: give a mini coding exercise
5. Review their code like a senior engineer
6. Only then move to the next concept

Never lecture for more than 3-4 sentences without asking a question.
Use real backend service examples (REST APIs, services, repositories).
